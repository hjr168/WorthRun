# 小程序内测检查清单

本清单用于《哪场值得跑》V0.1 真实数据内测。建议每轮内测前先完成后台数据检查，再做开发者工具、真机预览和体验版检查。

## 1. 后台准备

- [✅] 后台可以登录。
- [ ] 至少 10 条真实赛事已录入。
- [ ] 至少 5 条赛事已发布。
- [ ] 每条已发布赛事都有官方入口。
- [ ] 每条已发布赛事都有来源名称。
- [ ] 每条已发布赛事都有跑前判断。
- [ ] 每条已发布赛事至少 1 条判断理由。
- [ ] 每条已发布赛事有报名前确认清单。
- [✅] 隐藏、下架、草稿赛事不会出现在小程序。
- [✅] 操作日志正常记录。

## 2. 小程序功能

- [✅] 首页可以加载。
- [✅] 首页无偏好状态正常。
- [✅] 首页有偏好状态正常。
- [✅] 首页偏好无匹配时有兜底。
- [✅] 赛事列表可以筛选城市。
- [✅] 赛事列表可以筛选距离。
- [✅] 赛事列表可以筛选报名状态。
- [✅] 赛事列表可以筛选跑前判断。
- [✅] 赛事列表可以触底加载。
- [✅] 赛事详情可以打开。
- [ ] 有来源检查时间时，详情页显示“来源最近检查”和正确时间。
- [ ] 没有来源检查时间时，详情页显示“等待复核”。
- [ ] 存在开放告警时，详情页显示低干扰复核提示，主按钮仍为“前往官方确认”。
- [✅] 详情页有合规提示。
- [✅] 详情页没有报名直达文案。
- [✅] 前往官方确认可以复制链接并弹窗说明。
- [✅] 收藏成功。
- [✅] 取消收藏成功。
- [✅] 我的收藏正常展示。
- [ ] 收藏中的开放告警赛事显示“信息复核中”标签，长赛事名不遮挡。
- [✅] 偏好设置可以保存。
- [✅] 偏好设置可以重置。
- [✅] 反馈纠错可以提交。
- [✅] 后台可以看到反馈。
- [✅] 配速计算器正常。
- [✅] 赛前清单可以切换分组。

## 3. 真机测试

- [✅] 开发者工具本地可用。
- [ ] 真机预览可访问 API。
- [ ] 局域网 IP 配置正确。
- [ ] 手机和电脑在同一 Wi-Fi。
- [ ] 防火墙允许访问 API 端口。
- [ ] 体验版使用 HTTPS API。
- [ ] 微信公众平台已配置 request 合法域名。
- [ ] 不使用 localhost 提审。
- [ ] 不使用 HTTP 提审。

## 4. 合规检查

- [✅] 没有报名直达文案。
- [✅] 没有“保证报名成功”。
- [✅] 没有“官方认证推荐”等未经授权表述。
- [✅] 没有虚构官方链接。
- [✅] 没有评论区。
- [✅] 没有用户发帖。
- [✅] 没有引导分享奖励。
- [✅] 没有排行榜、积分、勋章。
- [✅] 详情页显示“AI 整理，仅供参考，报名以官方为准。”。
- [✅] 官方入口只作为确认来源，不做报名闭环。
- [ ] 小程序公开响应不包含告警差异、来源证据或管理员备注。
