# 小程序提审准备

本文档用于《哪场值得跑》V0.1 体验版和微信小程序提审前准备，不构成法律意见。实际类目、域名和隐私配置以微信公众平台当前规则为准。

## 1. 小程序定位说明

哪场值得跑是面向跑步爱好者的跑步赛事信息整理与赛前决策参考工具，帮助用户查看公开赛事信息、收藏关注赛事、记录跑赛偏好，并在报名前前往官方渠道确认信息。

## 2. 服务类目建议

具体服务类目需以微信公众平台当前可选类目为准。可优先查看体育、生活服务、工具、信息查询等相关类目，不要把本文档中的建议写成唯一答案。

## 3. 域名配置

- API 必须使用 HTTPS。
- 需要在微信公众平台配置 request 合法域名。
- 提审配置不要使用 localhost。
- 提审配置不要使用局域网 IP。
- 提审配置不要使用 HTTP。
- 体验版和正式版使用不同环境时，需要确认小程序配置与微信公众平台域名配置一致。

## 4. 用户数据说明

当前 V0.1：

- 不强制微信登录。
- 使用匿名 userKey 存储偏好、收藏和反馈。
- userKey 保存在小程序本地缓存。
- 不采集真实姓名。
- 不采集手机号。
- 不采集身份证。
- 不采集定位。
- 反馈内容由用户主动填写。

## 5. 隐私政策注意事项

提审前需要人工准备：

- 隐私政策。
- 用户数据用途说明。
- 反馈内容处理说明。
- 收藏和偏好数据说明。
- 联系方式或数据删除方式。

## 6. 官方入口说明

- 小程序不提供官方报名闭环。
- 不使用“立即报名”。
- “前往官方确认”仅复制官方链接。
- 用户需要到官方渠道确认信息。
- 赛事信息以官方为准。

## 7. 审核说明建议文案

可复制到微信审核备注，并按实际情况微调：

```text
本小程序为跑步赛事信息整理与赛前决策参考工具，不提供赛事报名交易服务。赛事详情页展示“AI 整理，仅供参考，报名以官方为准”提示；“前往官方确认”仅复制公开官方链接，用户需前往官方渠道确认赛事信息。当前版本不含社区、评论、用户发帖、付费报名、分享奖励等功能。
```

## 8. 审核前截图建议

- 首页。
- 赛事列表。
- 赛事详情。
- 赛事详情合规提示。
- 前往官方确认弹窗。
- 偏好设置。
- 我的收藏。
- 反馈纠错。
- 赛前清单。
- 配速计算器。
