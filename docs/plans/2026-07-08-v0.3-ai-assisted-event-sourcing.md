# V0.3 AI-Assisted Event Sourcing Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build an AI-assisted赛事源入库 workflow where configured sources produce event candidates, AI extracts structured draft data with evidence, and admins manually review,补充, and promote candidates into normal event drafts.

**Architecture:** Add a backend-only ingestion lane next to the existing event CRUD: `EventSource` defines where candidates come from, `EventCandidate` stores extracted structured data and evidence, and existing `Event` remains the only public-facing race record. AI may discover/extract candidate data, but it must not publish, must not bypass manual review, and must keep source URLs/evidence visible to admins. The miniapp remains unchanged in this phase.

**Tech Stack:** Node.js + TypeScript + Express + Prisma; React 18 + Ant Design 5 + Vite admin; OpenAI Responses API for structured extraction and optional web search; Zod/Vitest for schema validation and tests; robots.txt-aware HTTP fetching for configured page sources.

---

## Research Notes

- OpenAI Structured Outputs supports constraining model output to a JSON Schema with strict mode, which fits event extraction into a deterministic candidate schema.
- OpenAI Web Search can return up-to-date sourced results and expose searched URLs through sources metadata; use this only for optional source discovery, not as an unquestioned truth source.
- RFC 9309 defines robots.txt as instructions crawlers are requested to honor. For WorthRun, the crawler should fetch `/robots.txt`, respect disallow rules, set a clear user agent, and rate-limit requests.
- Current WorthRun data model already has `infoStatus = ai_generated`, `sourceLevel`, `fieldConfidence`, `publishStatus=draft`, operation logs, and manual publish gates. This is a good fit for AI-assisted drafts.

## Product Boundary Update

Replace the previous V0.3 guardrail:

```text
Do not add AI collection or AI parsing.
```

with:

```text
AI may collect candidate event information from explicitly configured sources and may parse it into structured candidate drafts. AI must not publish, must not create public miniapp-visible records directly, must not invent missing official links, and must preserve evidence for manual review.
```

Still preserve:

- Public copy: `AI 整理，仅供参考，报名以官方为准。`
- Official action text: `前往官方确认`
- No `立即报名` label.
- No automatic publishing.
- No official报名闭环.
- No scraping private/login-only pages.
- No importing placeholders as real data.

```typescript
import robotsParser from 'robots-parser';
import * as cheerio from 'cheerio';

const DEFAULT_USER_AGENT = process.env.AI_INGEST_USER_AGENT || 'WorthRunBot/0.1';
const MAX_HTML_BYTES = 800_000;

export interface FetchedPage {
  url: string;
  title: string;
  text: string;
}

export function normalizeAllowedDomains(domains: string[]) {
  return domains
    .map((domain) => domain.replace(/^https?:\/\//, '').replace(/\/$/, '').trim().toLowerCase())
    .filter(Boolean);
}

export function shouldAllowUrl(url: string, allowedDomains: string[]) {
  const parsed = new URL(url);
  const domains = normalizeAllowedDomains(allowedDomains);
  return domains.length === 0 || domains.some((domain) => parsed.hostname === domain || parsed.hostname.endsWith(`.${domain}`));
}

export async function fetchRobotsAllowedPage(url: string, allowedDomains: string[]): Promise<FetchedPage> {
  if (!shouldAllowUrl(url, allowedDomains)) {
    throw new Error('来源 URL 不在允许域名内');
  }

  const parsed = new URL(url);
  const robotsUrl = `${parsed.origin}/robots.txt`;
  const robotsResponse = await fetch(robotsUrl, { headers: { 'User-Agent': DEFAULT_USER_AGENT } }).catch(() => null);
  if (robotsResponse && robotsResponse.ok) {
    const robotsText = await robotsResponse.text();
    const robots = robotsParser(robotsUrl, robotsText);
    if (!robots.isAllowed(url, DEFAULT_USER_AGENT)) {
      throw new Error('robots.txt 不允许抓取该页面');
    }
  }

  const response = await fetch(url, { headers: { 'User-Agent': DEFAULT_USER_AGENT } });
  if (!response.ok) throw new Error(`来源页面请求失败：${response.status}`);
  const html = (await response.text()).slice(0, MAX_HTML_BYTES);
  const $ = cheerio.load(html);
  $('script,style,noscript,svg').remove();
  const title = $('title').text().trim();
  const text = $('body').text().replace(/\s+/g, ' ').trim().slice(0, 60_000);
  return { url, title, text };
}
```

```typescript
import OpenAI from 'openai';
import { aiEventCandidateSchema, normalizeAiCandidate } from './eventCandidateSchema';

const client = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });
const MODEL = process.env.AI_INGEST_MODEL || 'gpt-5.5';
export const AI_EVENT_PROMPT_VERSION = 'ai-event-candidate-v1';

export function buildExtractionPrompt(input: { sourceName: string; sourceUrl: string; text: string }) {
  return [
    '你是跑步赛事信息抽取助手，任务是从来源文本中抽取赛事候选草稿。',
    'AI 只生成候选草稿，不能发布赛事，不能替代人工核验。',
    '不要编造官方入口；来源没有明确官方入口时 officialUrl 必须为 null。',
    '每个关键字段必须给 evidence，evidence.quote 必须来自来源文本。',
    '赛事信息最终必须由后台管理员人工补充和确认。',
    `来源名称：${input.sourceName}`,
    `来源 URL：${input.sourceUrl}`,
    `来源文本：${input.text}`,
  ].join('\n\n');
}

export async function extractEventCandidate(input: {
  sourceName: string;
  sourceUrl: string;
  text: string;
}) {
  const response = await client.responses.create({
    model: MODEL,
    input: buildExtractionPrompt(input),
    text: {
      format: {
        type: 'json_schema',
        name: 'worth_run_event_candidate',
        strict: true,
        schema: zodToJsonSchemaShape(),
      },
    },
  });

  return normalizeAiCandidate(JSON.parse(response.output_text));
}

function zodToJsonSchemaShape() {

    additionalProperties: false,

      'eventDate',

      'signupDeadline',
      'officialUrl',

      'sourceUrl',
      'sourceLevel',

      'judgementSummary',
      'judgementReasons',
      'suitableFor',
      'notSuitableFor',
      'tags',
      'evidence',
      'confidence',

    properties: {
      eventName: { type: 'string' },
      city: { type: 'string' },
      eventDate: { type: ['string', 'null'], description: 'YYYY-MM-DD or null' },
      distanceItems: { type: 'array', items: { type: 'string' } },
      signupStatus: {
        type: 'string',
        enum: ['signup_open', 'closing_soon', 'closed', 'not_started', 'unknown'],
      },
      signupDeadline: { type: ['string', 'null'] },
      officialUrl: { type: ['string', 'null'] },
      sourceName: { type: 'string' },
      sourceUrl: { type: ['string', 'null'] },
      sourceLevel: { type: 'string', enum: ['official', 'trusted', 'secondary', 'unknown'] },
      runJudgement: { type: 'string', enum: ['priority', 'watch', 'unverified'] },
      judgementSummary: { type: 'string' },
      judgementReasons: { type: 'array', items: { type: 'string' } },
      suitableFor: { type: 'array', items: { type: 'string' } },
      notSuitableFor: { type: 'array', items: { type: 'string' } },
```

```typescript
export function buildCandidateFingerprint(eventName: string, city: string, eventDate: string | null) {
  return [eventName.trim(), city.trim(), eventDate || 'unknown-date'].join('|');
}

export async function runEventSource(sourceId: string) {
  const source = await prisma.eventSource.findUnique({ where: { id: sourceId } });
  if (!source) throw new Error('赛事源不存在');
  if (source.status !== 'active') throw new Error('赛事源未启用');
  if (!source.entryUrl) throw new Error('MVP 仅支持 page_url 赛事源');

  const page = await fetchRobotsAllowedPage(source.entryUrl, source.allowedDomains);
  const candidate = await extractEventCandidate({
    sourceName: source.name,
    sourceUrl: page.url,
    text: page.text,
  });

  const duplicate = candidate.eventDate
    ? await prisma.event.findFirst({
        where: {
          eventName: candidate.eventName,
          city: candidate.city,
          eventDate: new Date(`${candidate.eventDate}T00:00:00.000Z`),
        },
        select: { id: true },
      })
    : null;

  const saved = await prisma.eventCandidate.create({
    data: {
      sourceId,
      status: duplicate ? 'needs_review' : 'new',
      eventName: candidate.eventName,
      city: candidate.city,
      eventDate: candidate.eventDate ? new Date(`${candidate.eventDate}T00:00:00.000Z`) : null,
      sourceUrl: candidate.sourceUrl,
      officialUrl: candidate.officialUrl,
      extractedData: candidate as Prisma.InputJsonObject,
      evidence: candidate.evidence as Prisma.InputJsonArray,
      confidence: candidate.confidence as Prisma.InputJsonObject,
      duplicateEventId: duplicate?.id ?? null,
      aiModel: process.env.AI_INGEST_MODEL || 'gpt-5.5',
      aiPromptVersion: AI_EVENT_PROMPT_VERSION,
    },
  });

  await prisma.eventSource.update({
    where: { id: sourceId },
    data: { lastRunAt: new Date(), lastRunStatus: 'success' },
  });

  return saved;
}
```
